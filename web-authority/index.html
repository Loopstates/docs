12 Powerful Inspection Modules

🔍 SEO Metadata & SERP Preview

Inspect title, meta description, canonical tags, and robots directives. Live SERP snippet previews for both desktop and mobile truncation.

🏗️ Heading Structure Analysis

Visualize your full H1–H6 hierarchy, catch multiple H1s, detect skipped heading levels, and export the complete heading list.

🛡️ Page Status & Security

Detect HTTP status codes, redirect chains, mixed content, and validate SSL certificates, TLS protocols, and security headers like HSTS and CSP.

🔗 Link Analysis

Full audit of internal, external, nofollow, sponsored, and UGC links. Detect broken links, analyse anchor text diversity, and export any link list.

🖼️ Image SEO Audit

Find images missing ALT text, detect oversized files, check lazy loading, and identify modern formats (WebP / AVIF). Export the full image report.

📋 Structured Data (Schema)

Detect JSON-LD and Microdata, identify schema types, receive validation warnings, and check rich result eligibility at a glance.

📣 Social Metadata Validation

Preview how your page looks when shared on Facebook, LinkedIn, and X/Twitter. Validate OpenGraph completeness and social image dimensions.

⚡ Core Web Vitals & Performance

View LCP, INP, and CLS scores with threshold warnings. Get resource loading signals to pinpoint what's hurting your page speed.

🕷️ Crawl & Indexing Diagnostics

Access and interpret robots.txt rules, auto-discover sitemaps, and verify crawl directives and indexability — all without leaving the extension popup.

🤖 AI & LLM Readiness

Detect and validate llms.txt, check AI crawler accessibility, and preview how AI agents will extract content from your page.

📤 Export & Reporting

Export headings, link lists, image data, page text, or a full SEO audit report as PDF — all with a single click, directly from the popup.
